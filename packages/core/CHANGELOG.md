# Changelog

## 1.0.0 — 2026-05-14

Initial public release as `@justanothermldude/mcp-exec-oss-core`.

Open-source companion to `@justanothermldude/meta-mcp-core`. Generic MCP server connection pool, registry, and tool cache utilities. No gateway-specific auth (Cursor OAuth, PAT passthrough, backend-auth headers) — strictly generic env-var resolution.
